🆘 AYUDA RÁPIDA
================================================================================

Problema: "dukascopy_python not found"
  → pip install dukascopy-python

Problema: "No data available"
  → Puede ser fin de semana (mercado cerrado)
  → Ampliar DAYS_TO_DOWNLOAD

Problema: Script lento
  → Verificar internet
  → Reducir DAYS_TO_DOWNLOAD

Más ayuda:
  → python test_dukascopy.py
  → Lee MIGRACION_A_DUKASCOPY.md
